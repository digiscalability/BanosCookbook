## Integration Code Example

Add this to your `video-preview-modal.tsx`:

```tsx
const exportToCapCut = async (videoUrl: string) => {
  try {
    // Download video file
    const response = await fetch(videoUrl);
    const blob = await response.blob();

    // Create download link
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = 'recipe-video-capcut.mp4';
    a.click();

    URL.revokeObjectURL(url);

    toast({
      title: "Downloaded for CapCut!",
      description: "Open in CapCut app to edit your recipe video."
    });
  } catch (error) {
    toast({
      variant: "destructive",
      title: "Download failed",
      description: "Try right-clicking the video to save."
    });
  }
};
```

- **Export Quality Poor**: Use "Highest" quality setting
- **File Too Large**: Use CapCut's compression or shorten video

- Close other apps while editing
- Use CapCut's cloud storage for large projects
- Export in segments for very long videos
