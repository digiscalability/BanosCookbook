# Runway ML Prompt Length Fix

## 🐛 Issue Encountered

### Error Message
```
Error: 400 {"error":"Validation of body failed","issues":[{"origin":"string","code":"too_big","maximum":1000,"inclusive":true,"path":["promptText"],"message":"Too big: expected string to have <=1000 characters"}]}
```

### Root Cause
- **Problem**: Runway ML API has a **1000 character maximum** for the `promptText` parameter
- **What happened**: Video scripts generated by Gemini are often 1000+ characters long
- **Impact**: Video generation failed when trying to use the full script as the prompt

### Example
```
Original script length: 1,547 characters
Runway ML limit: 1,000 characters
Result: ❌ API error 400
```

---

## ✅ Solution Implemented

### Intelligent Prompt Optimization Function
Created `optimizePromptForRunway()` in `src/lib/openai-video-gen.ts`:

```typescript
function optimizePromptForRunway(recipeTitle: string, videoScript: string): string {
  // 1. Extract first 3 lines (the hook and key visuals)
  const lines = videoScript.split('\n').filter(line => line.trim().length > 0);
  let extractedScript = lines.slice(0, 3).join(' ').trim();

  // 2. Truncate if needed (max 500 chars for script content)
  if (extractedScript.length > 500) {
    extractedScript = extractedScript.substring(0, 500).trim();
  }

  // 3. Build prompt with cinematography instructions
  const basePrompt = `Recipe video: "${recipeTitle}". ${extractedScript}. Cinematic food photography, smooth camera movement, appetizing lighting, professional presentation.`;

  // 4. Final safety check - ensure ≤ 1000 characters
  if (basePrompt.length > 1000) {
    // Truncate more aggressively if needed
    const maxScriptLength = 1000 - `Recipe video: "${recipeTitle}". . Cinematic food photography, smooth camera movement, appetizing lighting.`.length;
    const truncatedScript = extractedScript.substring(0, maxScriptLength).trim();
    return `Recipe video: "${recipeTitle}". ${truncatedScript}. Cinematic food photography, smooth camera movement, appetizing lighting.`;
  }

  return basePrompt;
}
```

## 🎯 Why This Approach Works

### 1. **Preserves the Hook**
- The first 3 lines of video scripts typically contain:
  - The attention-grabbing hook
  - Key visual elements
  - Main recipe concept
- This is the most important content for video generation

### 2. **Adds Professional Context**
- Appends cinematography instructions:
  - "Cinematic food photography"
  - "Smooth camera movement"
  - "Appetizing lighting"
  - "Professional presentation"
- Ensures Runway ML understands the desired style

### 3. **Guaranteed Compliance**
- Multiple truncation levels ensure prompt never exceeds 1000 chars
- Safety check at the end validates length
- Throws error if somehow still too long (fail-safe)

### 4. **No User Action Required**
- Happens automatically during video generation
- Users don't need to edit scripts
- Transparent and reliable
